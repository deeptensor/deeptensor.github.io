Содержание выполняемых работ


  1. Исследование возможностей и перспектив использования различных тензорных сетей для малорангового (сжатого) представления данных и разработка новых алгоритмов для тензорных разложений, в том числе, для разложения тензорного поезда (tensor train) и квантизованного разложения тензорного поезда (quantized tensor train).
    [ Ответственный исполнитель : И. В. Оселедец ].
  2. Исследование методов оптимизации на тензорных многообразиях для решения задач минимизации функционалов в рамках малорангового тензорного разложения, в том числе, для эффективного построения малорангового представления тензорной сети.
    [ Ответственный исполнитель : И. В. Оселедец ].
  3. Разработка метода обучения глубинных байесовских генеративных моделей с использованием обобщенной K-выборочной вариационной нижней оценки на маргинальное правдоподобие.
    [ Ответственный исполнитель : Д. П. Ветров ].
  4. Разработка стохастического метода оптимизации второго порядка для решения невыпуклых задач оптимизации.
    [ Ответственный исполнитель : Д. П. Ветров ].
  5. Создание нового набора данных для тестирования и обучения глубинных нейронных сетей.
    [ Ответственный исполнитель : В. С. Лемпицкий ].
  6. Разработка и валидация методов синтеза глубинного обучения и классических методов компьютерного зрения (в том числе, каскадирования, преобразования Хафа, интегральных изображений) для быстрой фильтрации изображений.
    [ Ответственный исполнитель : В. С. Лемпицкий ].
  7. Закупка вычислительного оборудования и комплектующих для проведения научных исследований.
  8. Участие членов научного коллектива лаборатории в международных конференциях, семинарах и симпозиумах по направлениям тензорных методов, машинного обучения и интеллектуального анализа данных (не менее пяти мероприятий).
  9. Командировки членов научного коллектива в ведущие мировые научно-образовательные центры с целью стажировки и/или проведения совместных научных исследований по направлениям тензорных методов, машинного обучения и интеллектуального анализа данных (не менее одной командировки).
  10. Разработка и валидация новых методов синтеза двухмерных изображений с помощью глубинных нейронных сетей.
    [ Ответственный исполнитель : В. С. Лемпицкий ].
  11. Закупка материалов и комплектующих для проведения научных исследований.
  12. Организационно-техническое сопровождение проекта, подготовка помещений, необходимых для функционирования лаборатории, размещение вычислительного оборудования и установка комплектующих, закупаемых в рамках соглашения по гранту и необходимых для проведения исследований.
  13. Подготовка и проведение новых и/или модернизированных лекционных курсов, семинарских занятий и иных образовательных мероприятий по теме исследований для студентов и аспирантов Сколтеха (не менее четырех образовательных курсов).
  14. Командировки членов научного коллектива лаборатории в ведущие мировые научные центры с целью обмена опытом и/или проведения совместных научных исследований по направлениям тензорных методов, машинного обучения и интеллектуального анализа данных (не менее одной командировки).

  1. Анализ архитектуры современных глубинных нейронных сетей (сверточные и рекуррентные нейронные сети, ограниченные машины Больцмана, автокодировщики и т.д.) и выявление их связи с тензорными сетями. Применение тензорных факторизаций и новых разработанных в лаборатории методов оптимизации на тензорных многообразиях для задач сжатия и обучения глубинных нейронных сетей.
    [ Ответственный исполнитель : И. В. Оселедец ].
  2. Построение новых типов слоев для глубинных нейронных сетей на основе формализма метода малоранговых тензорных факторизаций.
    [ Ответственный исполнитель : И. В. Оселедец ].
  3. Разработка метода получения приближенного апостериорного распределения из совокупности K предложных плотностей, настроенных на данную генеративную модель.
    [ Ответственный исполнитель : Д. П. Ветров ].
  4. Эффективная программная реализация разработанных алгоритмов обучения и вывода для параллельной обработки данных на графических ускорителях. Разработка методов деблюрринга (устранения размытостей и искажений) изображений на основе глубинного обучения.
    [ Ответственные исполнители : И. В. Оселедец, В. С. Лемпицкий, Д. П. Ветров ].
  5. Разработка метода, ускоряющего нахождение объектов на изображении за счет привлечения сторонней информации (например, данных о глубине) и/или использования новых нейросетевых архитектур.
    [ Ответственный исполнитель : В. С. Лемпицкий ].
  6. Организация на базе Сколтеха международной научной конференции по направлениям тензорных методов, машинного обучения и интеллектуального анализа данных.
  7. Закупка комплектующих для оборудования для проведения научных исследований.
  8. Участие членов научного коллектива лаборатории в международных конференциях, семинарах и симпозиумах по направлениям тензорных методов, машинного обучения и интеллектуального анализа данных (не менее шести мероприятий).
  9. Командировки членов научного коллектива лаборатории в ведущие мировые научно-образовательные центры с целью стажировки и/или проведения совместных научных исследований по направлениям тензорных методов, машинного обучения и интеллектуального анализа данных (не менее одной командировки).
  10. Разработка и валидация новых методов синтеза трехмерных данных с помощью глубинных нейронных сетей.
    [ Ответственный исполнитель : В. С. Лемпицкий ].
  11. Закупка материалов и комплектующих для проведения научных исследований.
  12. Организационно-техническое сопровождение проекта.

  15. Командировки членов научного коллектива лаборатории в ведущие мировые научные центры с целью обмена опытом и/или проведения совместных научных исследований по направлениям тензорных методов, машинного обучения и интеллектуального анализа данных (не менее одной командировки).

Разработка комбинированных архитектур, включающих тензоризованные глубинные нейронные сети, графические модели, стохастическую оптимизацию и байесовский вывод.
