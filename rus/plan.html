План работ на ближайшие три года


  1. Исследование возможностей и перспектив использования различных тензорных сетей для малорангового (сжатого) представления данных и разработка новых алгоритмов для тензорных разложений, в том числе, для разложения тензорного поезда (tensor train) и квантизованного разложения тензорного поезда (quantized tensor train).

    [ 2017. Ответственный исполнитель : И. В. Оселедец ]

  2. Исследование методов оптимизации на тензорных многообразиях для решения задач минимизации функционалов в рамках малорангового тензорного разложения, в том числе, для эффективного построения малорангового представления тензорной сети.

    [ 2017. Ответственный исполнитель : И. В. Оселедец ]

  3. Разработка метода обучения глубинных байесовских генеративных моделей с использованием обобщенной K-выборочной вариационной нижней оценки на маргинальное правдоподобие.

    [ 2017. Ответственный исполнитель : Д. П. Ветров ]

  4. Разработка стохастического метода оптимизации второго порядка для решения невыпуклых задач оптимизации.

    [ 2017. Ответственный исполнитель : Д. П. Ветров ]

  5. Создание нового набора данных для тестирования и обучения глубинных нейронных сетей.

    [ 2017. Ответственный исполнитель : В. С. Лемпицкий ]

  6. Разработка и валидация методов синтеза глубинного обучения и классических методов компьютерного зрения (в том числе, каскадирования, преобразования Хафа, интегральных изображений) для быстрой фильтрации изображений.

    [ 2017. Ответственный исполнитель : В. С. Лемпицкий ]

  7. Закупка вычислительного оборудования и комплектующих для проведения научных исследований.

    [ 2017. ]

  8. Участие членов научного коллектива лаборатории в международных конференциях, семинарах и симпозиумах по направлениям тензорных методов, машинного обучения и интеллектуального анализа данных (не менее пяти мероприятий).

    [ 2017. ]

  9. Командировки членов научного коллектива в ведущие мировые научно-образовательные центры с целью стажировки и/или проведения совместных научных исследований по направлениям тензорных методов, машинного обучения и интеллектуального анализа данных (не менее одной командировки).
